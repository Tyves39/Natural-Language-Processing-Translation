# Natural Language Processing & Machine Translation

This directory explores advanced text processing techniques, from sentiment analysis using Convolutional Neural Networks (CNN) to neural machine translation with Attention mechanisms.

## Projects

* **Sentiment Analysis (TextCNN)**: Two implementations (MindSpore & TensorFlow) for movie review classification, featuring text vectorization and convolutional feature extraction.
* **Neural Machine Translation**: An Encoder-Decoder model with Attention for Spanish-to-English translation.
* **Text Preprocessing Pipeline**: Custom cleaning scripts using Regex, Tokenization, and Vocabulary management.

## Technical Skills
* **Frameworks**: TensorFlow/Keras, MindSpore.
* **Deep Learning**: CNNs for Text, RNNs, Encoder-Decoder architectures, Attention Mechanisms.
* **NLP Tools**: Word Embeddings, Padding, Tokenization, Unicode processing.
